Password Management

It is important to securely store, generate and manage passwords.

There are many online password management application that provide convenience. However, there could be some disadvatages:

  • Dependency on Internet: These applications require Internet access. If for some reason you Internet is down, you cannot access your passwords.
  • Security Risks: Your passwords can be accessed by a third-party service.
  • Data Breaches: Unauthorized parties access to the sensitive data in the servers.(
  • Cost: There are some applications free, but those that offer robust security could be expensive.

Instead of using an online password management application, we can use a tool that allows to store an encrypted file with the password we need in a desired location.

KeePassXC is a secure and open-sources password manager that can help to store passwords.

KeePassXC

  • Download KeePassXC
  • Go to the folder where KeePassXC was downloaded, right-click on the icon, click on Permission, select the checkbox "Allow executing file as program". Close the properties window.
  • Now, you can excecute KeePassXC. Double click on the icon.
  • You can create a new database for your passwords, adjust the database encryptation, add your passwords, and save database (file) with your passwords.
